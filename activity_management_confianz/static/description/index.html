Before you start

What you'll need.

Configure user permissions before setting up your teams and activity tracking.

  • Developer Mode Activated
  • Manage Teams Permission
  • Team Leader Assignment

How to enable User Permissions

Navigate to Settings

Go to Settings → Users & Companies → Users.

Select your User

Open the user record you wish to grant management rights to.

Locate Manage Teams

Find the checkbox in the user access rights tab.

Enable the group

Click to enable Manage Teams to allow team creation.

Save changes

Press Save to apply the new permissions immediately.

Permission Setting
Technical Settings / Manage Teams / Enabled

Once enabled, this user will have full access to the Activity Teams creation menu.

Configuration

Set up your first team in Odoo.

Organize users and assign leaders in just a few clicks to start tracking collective productivity.

Navigation

Go to the Activity Teams menu item.

Create

Click the Create button to start a new team.

Assign Leader

Set a user as the primary Team Leader.

Add Members

Select multiple users to join the team list.

Save

Save the record to initialize the team structure.

Verify

The team will now appear in activity scheduling dropdowns.
